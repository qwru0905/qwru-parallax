## 페이지 설계

### `/` — Home (허브)

- **상단:** Navbar (로고 `qwru0905` + 네비게이션 링크)
- **히어로 영역:** "qwru0905" 타이틀, 부제 "개발자 · 유튜버 · 작곡가"
- **포털 카드 3개:** Dev / YouTube / Music 각각 카드로 표시, 클릭 시 해당 페이지로 이동
  - Dev 카드: 보라 액센트
  - YouTube 카드: 빨강 액센트
  - Music 카드: 파랑 액센트 + "준비 중" 표시

---

### `/dev` — Developer

- **자기소개:** 짧은 텍스트 소개
- **기술 스택:** 태그(pill) 형태로 나열
- **프로젝트 리스트:** 프로젝트명 + 한 줄 설명 + GitHub 링크
- **GitHub 링크:** 페이지 상단 또는 소개 영역에 GitHub 프로필 링크 연결

---

### `/youtube` — YouTube

- **채널 소개:** 채널명, 구독자 수
- **구독하기 버튼:** YouTube 채널로 외부 링크
- **최신 영상 그리드:** 초기에는 영상 썸네일 + 링크를 정적으로 하드코딩. 추후 YouTube Data API v3로 동적 교체 가능한 구조로 작성
- **채널 테마:** 종합 게임 채널

---

### `/music` — Music

- **Coming Soon 레이아웃:** 음표 아이콘, "음악을 준비하고 있어요" 메시지
- **부연 설명:** 곧 채워질 것임을 암시하는 짧은 문구
- **애니메이션:** 펄스 도트 등 소소한 생동감
